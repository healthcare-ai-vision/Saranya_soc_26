# Week 3-4 Submission

## Project Title
Brain Tumor Detection using YOLOv8

## Objective
Train an object detection model using Roboflow dataset and YOLOv8.

## Dataset
Brain Tumor Computer Vision Dataset from Roboflow Universe

## Tools Used
- Roboflow
- Ultralytics YOLOv8
- Google Colab
- Python

## Training Details
- Model: YOLOv8 Nano
- Epochs: 10
- Hardware: Google Colab T4 GPU

## Result
The trained model successfully detected brain tumors in MRI scan images.

Prediction results were saved in:
`/content/runs/detect/predict`
